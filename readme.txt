Switch the theme by the user agent. Can be specified user agent by a regular expression.

== Description ==

* Switch the theme by the user agent.
* Can be specified user agent by a regular expression.
* Can add types and set the corresponding user agent.
* Sibling plugin -> [UserAgent Content Switcher](https://wordpress.org/plugins/useragent-content-switcher/).

== Installation ==

1. Upload `useragent-themes-switcher` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the Settings page, set the theme and the corresponding user agent.
